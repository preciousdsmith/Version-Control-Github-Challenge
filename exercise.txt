1. Explain what the terminal is in your own words.
A terminal is an app or program on a Mac operating system that allows a user to send commands for a computer. 

2. What is the major difference between Git and GitHub?
GitHub is a Git repository hosting service. GitHub manages repositories. Git is a widely used version control system. Git also allows a user to keep track of code and changes.

3. What are the steps to fork a GitHub repository? In order to fork a GitHub repository, a user must click the fork icon near the top right corner of the GitHub page. Second, "Owner" will appear on the screen, and the user will be required to select an owner for the repository they want to fork. A user may add a description and a field name to the fork. After those optional steps, the user will press "create fork."
